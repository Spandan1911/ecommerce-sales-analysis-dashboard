# ecommerce-sales-analysis-dashboard
Interactive Ecommerce Sales Analysis Dashboard built in Microsoft Excel to analyze sales, profit, orders, quantity, category performance, regional sales trends, and business KPIs through dynamic visualizations and slicers.

# Project Overview

This project presents an interactive Ecommerce Sales Analysis Dashboard developed in Microsoft Excel.
The dashboard provides a comprehensive view of business performance by analyzing sales, profit, orders, quantities sold, category-wise performance, regional sales distribution, and customer segments.
The objective is to transform raw sales data into meaningful business insights that support data-driven decision-making.
